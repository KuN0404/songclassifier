# Songclassifier

Web ini dibuat untuk memprediksi genre sebuah lagu berdasarkan lirik lagu

> #### Installation
1. Siapkan Virtual Env
    ```bash
    python -m venv venv
    source venv/scripts/active

2. Install requrements
    ```bash
    pip install -r requirements.txt

3. Jalankan Python
    ```bash
    python app.py
